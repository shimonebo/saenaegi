# Safe Way 팀 백엔드 연동

## 적용되는 내용

- 팀 백엔드의 `POST /route/safe` 호출
- 장소명 데모 좌표 변환
- 백엔드 응답을 현재 결과 UI 형식으로 변환
- 안전 경로와 최단 경로 비교 표시
- 로딩 및 오류 화면
- `osmnx`, `networkx` 설치 항목 추가

## 데모 입력값

```text
출발지: 대전서원초등학교
도착지: 둔산동 보라아파트
```

또는 입력창에 직접 다음처럼 좌표를 넣을 수 있습니다.

```text
127.3760,36.3500
```

## 중요

현재 팀 백엔드는 장소명이 아니라 `경도,위도` 좌표를 받습니다.
이 연동 코드는 정해진 데모 장소명을 좌표로 바꿔 호출합니다.

임의의 모든 장소명을 지원하려면 이후 카카오 로컬 검색 또는
다른 지오코딩 API를 추가해야 합니다.
